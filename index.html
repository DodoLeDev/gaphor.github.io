{% capture about %} Gaphor is a UML modeling application written in Python. It is designed to be easy to use, while still being powerful. Gaphor implements a fully-compliant UML 2 data model, so it is much more than a picture drawing tool. You can use Gaphor to quickly visualize different aspects of a system as well as create complete, highly complex models. {% endcapture %} {% include services.html id="about" title="About" description=about services=site.usps %} {% capture download %} There are many ways to install Gaphor. The simplest is to download the official installer for Windows or macOS. For Linux you can install Gaphor using FlatHub. You can also use Python's built-in pip tool as long as you have all of the required dependencies installed. {% endcapture %}
