About

Simple.

Gaphor is a simple and easy to use modeling tool for UML, we believe complexity impedes a novice modeler's first steps.

Aimed at Beginners.

It is aimed at beginning modelers who want a simple and fast tool so that they can focus on learning.

Less is More.

Gaphor has only the most essential features, so users are not intimidated by a baffling interface.

One Way.

There should be one—and preferably only one—obvious way to do it with Gaphor.

Have fun!

Learning should inspire fun - Gaphor helps learners quickly create models of software and systems.

Not Enterprise.

It is not a full featured enterprise tool.
