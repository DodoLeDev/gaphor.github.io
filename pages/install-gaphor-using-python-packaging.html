Install Gaphor Using Python Packaging

If you have the latest stable version of Python installed and the Gaphor dependencies installed, you can also install Gaphor using a wheel from PyPI.

If you don't have the latest stable version of Python and the Gaphor dependencies installed, follow the development environment setup instructions section and instead of cloning the repository, create a virtual environment, and then execute the following:

$ pip install gaphor
$ gaphor
